About the Application

Poets4Life_Bot is an internal, proprietary automation tool developed exclusively for the Poets4Life media project. Powered by n8n workflows, this application is designed to streamline and automate the deployment of our own video content to our official TikTok profile.

This application is not intended for public use and is strictly restricted to authorized system administrators of the Poets4Life project.
